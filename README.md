# 📝 Advanced To-Do List Application

A feature-rich desktop-based task management application built with Python's Tkinter library. Organize your tasks efficiently with priorities, due dates, search filters, color themes, and real-time statistics.

---

## ✨ Features

### Core Features
- ✅ Add, Edit, and Delete Tasks
- ✅ Mark Tasks as Complete or Incomplete
- ✅ Automatic Task Saving
- ✅ Persistent JSON Storage

### Advanced Features
- 📅 Due Dates for Tasks
- ⚠️ Priority Levels (High, Medium, Low)
- 🔍 Real-Time Search Functionality
- 📌 Smart Task Filters
- 🎨 Multiple Color Themes
- 📊 Statistics Dashboard
- 🖱️ Right-Click Context Menu
- ⌨️ Keyboard Shortcuts

## 📖 Project Description

The Advanced To-Do List Application is a desktop-based task management system developed using Python and Tkinter. It enables users to efficiently create, edit, organize, and track tasks through features such as priority management, due dates, search filters, statistics, and customizable themes.

The application uses JSON-based storage to save task data, ensuring that all tasks remain available even after closing the program.
